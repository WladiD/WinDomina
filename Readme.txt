- Durch Doppeltipp auf [CapsLock] wird der Win-Dominate-Modus aktiviert
- Durch erneuten Doppeltipp auf [CapsLock] oder einen einfach Tastendruck 
  auf [Escape] wird der Win-Dominate-Modus wieder verlassen
- Im akivierten Win-Dominate-Modus gilt folgendes Verhalten:
  - Die Zahlen 1-9 auf dem Nummernblock stellen das 3x3-Raster des 
    Bildschirms dar. Alternativ werden auch die normalen Zahlen akzeptiert,
    was aber deutlich an der Intuition verliert.
  - Wird eine Zahl [1]-[9] und anschließend [Enter] gedrückt, so wird das aktuell 
    aktive Fenster dahin positioniert.
  - Zusammenhängende Bereiche im 3x3-Raster können kombiniert werden. 
    Wenn z.B. ein Fenster das obere Drittel des Monitors einnehmen soll, 
    so drückt man [7] [8] [9] und bestätigt mit [Enter]. Das gleiche Ergebnis
    erreicht man auch durch [7] [9] [Enter]
  - Wurde eine Eingabe von einer oder mehreren Zahlen eingegeben, aber noch nicht
    mit [Enter] bestätigt, so kann die Angabe mit [-] auf dem Nummernblock oder mit 
    [Escape] abgebrochen werden
  - Wurde zuvor ein Fenster in einem Raster-Bereich positioniert und man möchte 
    dieses Fenster aktivieren, so drückt man 2 mal die entsprechende Zahl und 
    bestätigt mit [Enter]. Wurden bereits mehrere Fenster in einem Raster-Bereich
    positioniert so drückt man entsprechend weiter auf die entsprechende Zahl 
    bis das gewünschte Fenster sichtbar ist und bestätigt abschließend mit [Enter]
  - Das 3x3-Raster bezieht sich immer auf den Monitor in dem sich das aktuell
    aktive Fenster befindet. Mit der [0]-Taste wechselt man das Raster zwischen 
	den verfügbaren Monitoren und kann so das jeweilige Fenster schnell auf einen 
	bestimmten Monitor in die gewünschte Position bringen.
